# 🪐 Planet-RL Sandbox

> A procedural planet generation and orbital mechanics sandbox for training reinforcement learning agents on orbit insertion — across any planet in the universe.

![Sandbox Overview](sandbox_overview.png)

---

## What is this?

**Planet-RL** is a physics-accurate simulation environment for training RL agents to perform orbital insertion manoeuvres around procedurally generated planets. Every planet feature — atmosphere, terrain, magnetic field, gravity harmonics, moons — is individually toggleable, letting you control exactly how hard or complex the environment is.

The goal: train a single agent that can insert into orbit around *any* planet it has never seen before, regardless of gravity, atmosphere, or size.

---

## Features

### 🌍 Procedural Planet Generation
- Randomised radius, mass, and density within configurable ranges
- Gravity derived directly from mass and size — no manual tuning
- Seeded generation for reproducibility
- 5 real-world presets: **Earth, Mars, Venus, Moon, Titan**
- Batch generation for curriculum training

### 🌬️ Atmosphere System (toggleable)
7 atmosphere compositions with physically accurate density, pressure, and temperature profiles:

| Composition | Based On | Surface Density |
|---|---|---|
| `EARTH_LIKE` | Earth | 1.225 kg/m³ |
| `CO2_THIN` | Mars | 0.015 kg/m³ |
| `CO2_THICK` | Venus | 65.0 kg/m³ |
| `METHANE` | Titan | 5.3 kg/m³ |
| `NITROGEN` | Inert Earth-like | 1.16 kg/m³ |
| `HYDROGEN` | Gas giant envelope | 0.09 kg/m³ |
| `NONE` | Vacuum | — |

Exponential density model: `ρ(h) = ρ₀ · exp(-h / H)`

Includes wind speed, wind direction, and drag coefficient scaling.

### 🏔️ Terrain System (toggleable)
5 terrain archetypes with configurable elevation range and roughness:
`FLAT` · `CRATERED` · `MOUNTAINOUS` · `OCEANIC` · `VOLCANIC`

### 🧲 Magnetic Field (toggleable)
- 4 strength levels: `NONE`, `WEAK`, `MEDIUM`, `STRONG`
- Configurable dipole tilt angle
- Optional radiation belt modelling (inner + outer belts)

### 🪐 J2/J3 Oblateness (toggleable)
Gravity harmonics for more realistic orbital perturbations. Oblateness scales with rotation rate — fast rotators become more oblate automatically.

### 🌙 Moons (toggleable)
N-body moon perturbations with configurable count, mass fraction, and orbital radius.

## Usage

### Use a preset planet
```python
from core import PRESETS

earth = PRESETS["earth"]()
mars  = PRESETS["mars"]()

print(earth.summary())
# ═══ Earth ═══
#   Radius          : 6371.0 km  (1.000 R⊕)
#   Surface gravity : 9.820 m/s²
#   Escape velocity : 11.186 km/s
#   Atmosphere      : ON | ρ₀=1.225 kg/m³ | H=8.5 km
#   ...
```

### Generate a random planet
```python
from core import PlanetGenerator

gen = PlanetGenerator(seed=42)

# All features on
planet = gen.generate(
    name="Zephyria",
    atmosphere_enabled=True,
    terrain_enabled=True,
    magnetic_field_enabled=True,
    oblateness_enabled=True,
    moons_enabled=True,
)

# Airless rock — atmosphere and moons off
barren = gen.generate(
    name="Barren-IV",
    atmosphere_enabled=False,
    terrain_enabled=True,
    magnetic_field_enabled=False,
    oblateness_enabled=False,
    moons_enabled=False,
)
```

### Batch generation for curriculum training
```python
# 1000 diverse planets, all with atmospheres
planets = gen.batch(1000, atmosphere_enabled=True, oblateness_enabled=True)
```

### Query orbital mechanics
```python
planet = PRESETS["mars"]()

# Circular orbit speed and period at 300 km
v = planet.circular_orbit_speed(300_000)   # m/s
T = planet.circular_orbit_period(300_000)  # s

# Hohmann transfer delta-v: 200 km → 500 km
dv1, dv2 = planet.hohmann_delta_v(200_000, 500_000)

# Atmosphere density at 80 km altitude
rho = planet.atmosphere.density_at_altitude(80_000)

# Gravity at altitude including J2 correction
ax, ay, az = planet.gravity_vector_J2((r_x, r_y, r_z))
```

### Run a physics simulation
```python
import numpy as np
from core import OrbitalIntegrator, ThrusterConfig, AeroConfig, SpacecraftState, PRESETS

planet = PRESETS["mars"]()

integrator = OrbitalIntegrator(
    planet=planet,
    thruster=ThrusterConfig(max_thrust=3000, Isp=320),
    aero=AeroConfig(enabled=True, Cd=2.2, reference_area=10.0),
)

# Start on hyperbolic approach
initial = SpacecraftState(
    x=planet.radius + 500_000, y=0, z=0,
    vx=0, vy=4200, vz=0,      # slightly above orbital speed
    mass=2000, dry_mass=500,
)

# Retrograde burn for 3 minutes at T=0
schedule = [(0, 180, np.array([0, -2500, 0]))]   # (t_start, t_end, thrust_vec [N])

history = integrator.propagate(initial, duration=7200, dt=5, thrust_schedule=schedule)
```

### Use the Gymnasium RL environment
```python
from core import OrbitalInsertionEnv

env = OrbitalInsertionEnv(
    randomize_planet=True,        # new planet every episode
    atmosphere_enabled=True,      # include atmospheric drag
    oblateness_enabled=False,     # start simple
    moons_enabled=False,
    target_altitude=300_000,      # 300 km target orbit
    max_thrust=500,
    Isp=320,
    dt=10,
    max_steps=2000,
)

obs, info = env.reset()
print(info["planet_summary"])     # which planet did we get?

done = False
while not done:
    action = env.action_space.sample()     # replace with your agent
    obs, reward, terminated, truncated, info = env.step(action)
    done = terminated or truncated

print("Success:", info["success"])
print("Final altitude:", info["altitude_m"] / 1e3, "km")
```

## RL Environment Reference

### Observation Space — 10 dimensions
| Index | Variable | Range | Description |
|---|---|---|---|
| 0 | `altitude_norm` | 0 – ~2 | altitude / target altitude |
| 1 | `speed_norm` | 0 – ~2 | speed / target circular speed |
| 2 | `flight_path_angle` | −1 – 1 | FPA / π |
| 3 | `eccentricity` | 0 – 2 | current orbital eccentricity |
| 4 | `fuel_fraction` | 0 – 1 | remaining fuel / initial fuel |
| 5 | `heat_norm` | 0 – 1 | accumulated heat / heat limit |
| 6 | `planet_radius_norm` | 0 – 4 | planet radius / R⊕ |
| 7 | `surface_grav_norm` | 0 – ~5 | surface gravity / 9.81 |
| 8 | `atm_density_norm` | 0 – 10 | atm density at altitude / 1.225 |
| 9 | `target_alt_norm` | 0 – ~1 | target altitude / planet radius |

### Action Space — 3 continuous dimensions (all in [−1, 1])
| Index | Variable | Maps To |
|---|---|---|
| 0 | `thrust_magnitude` | 0 → max_thrust [N] |
| 1 | `pitch` | −π/2 → π/2 [rad] |
| 2 | `yaw` | −π → π [rad] |

### Reward Function
```
r = −|alt_norm − 1| × 0.5       # altitude error
  + −|speed_norm − 1| × 0.5     # speed error
  + −eccentricity × 0.3         # orbit shape
  + −fuel_used × 0.01           # efficiency
  + −0.01                       # step penalty
  + +100  (on success)          # orbit achieved
  + −50   (on crash)
```

Success condition: altitude within 5% of target AND eccentricity < 0.05.

### Terminal Conditions
| Condition | Type |
|---|---|
| Orbit achieved | `terminated = True` |
| Crashed into surface | `terminated = True` |
| Escaped sphere of influence | `terminated = True` |
| Heat limit exceeded | `terminated = True` |
| Max steps reached | `truncated = True` |
| Out of fuel | `truncated = True` |

## Physics Models

### Gravity
- **Point mass:** `g = μ/r²` where `μ = G·M`
- **J2 correction:** full oblateness perturbation in ECI frame when `oblateness_enabled=True`

### Atmosphere
- **Density:** exponential scale height model `ρ(h) = ρ₀·exp(−h/H)`
- **Temperature:** linear tropospheric lapse, isothermal above tropopause
- **Drag:** `F = ½ρv²·Cd·A`, deceleration `a = F/m`
- **Aeroheating:** Sutton-Graves approximation `q = k·√ρ·v³`

### Propulsion
- Tsiolkovsky rocket equation: `ṁ = −T / vₑ` where `vₑ = Isp·g₀`
- Thrust vector resolved in RTN (Radial-Transverse-Normal) frame

### Integrator
- Fixed-step **RK4** (default) or adaptive **RK45**
- State vector: `[x, y, z, vx, vy, vz, mass, heat_load]`

## Recommended RL Algorithms

| Algorithm | Why |
|---|---|
| **SAC** (Soft Actor-Critic) | Best for continuous thrust control, sample efficient |
| **TD3** | Stable alternative to SAC, deterministic policy |
| **PPO** | Good baseline, interpretable, easy to tune |
| **SAC + HER** | Add Hindsight Experience Replay for sparse reward cases |

For generalisation across planets, consider **MAML** or **PEARL** (meta-RL) — the planet parameters in the observation vector serve as task context.

## Curriculum Strategy

Train progressively by toggling features on as the agent improves:

```
Stage 1  →  Fixed Earth-like planet, no atmosphere, circular target orbit
Stage 2  →  Vary planet mass ±20%, add thin atmosphere
Stage 3  →  Full atmosphere + aerobraking, randomise composition
Stage 4  →  Enable oblateness (J2), elliptical target orbits
Stage 5  →  Full random planet generation, all features on
Stage 6  →  Multi-manoeuvre missions (transfer orbit → insertion → landing)
```
